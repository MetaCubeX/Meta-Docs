# Сторонние инструменты/клиенты
Инструменты/клиенты ниже используют или включают ядро mihomo. Мы **не контролируем напрямую** разработку этих сторонних инструментов/клиентов; в них могут отсутствовать новейшие функции и исправления из официальной версии. По любым проблемам, вызванным **не ядром**, обращайтесь в соответствующие проекты сторонних инструментов/клиентов.
