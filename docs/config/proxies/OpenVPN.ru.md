# OpenVPN

```{.yaml linenums="1"}
proxies

- name: "openvpn"
  type: openvpn
  server: vpn.example.com
  port: 1194
  proto: udp      
  # dev: tun      
  # cipher: AES-128-GCM 
  # auth: SHA256  
  ca: |
    -----BEGIN CERTIFICATE-----
    MIIB...example
    -----END CERTIFICATE-----
  cert: |
    -----BEGIN CERTIFICATE-----
    MIIB...example
    -----END CERTIFICATE-----
  key: |
    -----BEGIN PRIVATE KEY-----
    MIIE...example
    -----END PRIVATE KEY-----
  tls-crypt: |
    -----BEGIN OpenVPN Static key V1-----
    00000000000000000000000000000000
    ...
    -----END OpenVPN Static key V1-----
  udp: true
  # mtu: 1500
  # dialer-proxy: "ss1"
  # remote-dns-resolve: true      
  # dns: [ 1.1.1.1, 8.8.8.8 ]

```

[Общие поля](./index.md)

## server

**Обязательно.** Адрес сервера OpenVPN.

## port

**Обязательно.** Порт сервера OpenVPN, по умолчанию `1194`.

## proto

**Опционально.** Тип протокола, поддерживаются `udp` или `tcp`, по умолчанию `udp`.

## dev

**Опционально.** Тип виртуальной сетевой карты OpenVPN, в данный момент поддерживается только `tun`, по умолчанию `tun`.

## cipher

**Опционально.** Метод шифрования, в данный момент поддерживается только `AES-128-GCM`, по умолчанию `AES-128-GCM`.

## auth

**Опционально.** Метод аутентификации, в данный момент поддерживается только `SHA256`, по умолчанию `SHA256`.

## ca

**Обязательно.** Содержимое сертификата CA. Скопируйте из тега `<ca>` в файле `.ovpn`, исключая сами теги `<ca>`.

## cert

**Обязательно.** Содержимое клиентского сертификата. Скопируйте из тега `<cert>` в файле `.ovpn`, исключая сами теги `<cert>`.

## key

**Обязательно.** Содержимое закрытого ключа клиента. Скопируйте из тега `<key>` в файле `.ovpn`, исключая сами теги `<key>`.

## tls-crypt

**Опционально.** Ключ шифрования TLS. Скопируйте из тега `<tls-crypt>` в файле `.ovpn`, исключая сами теги `<tls-crypt>`.

## udp

**Опционально.** Использовать ли протокол UDP. `true` для UDP, `false` для TCP.

## mtu

**Опционально.** Максимальная единица передачи (MTU), по умолчанию `1500`.

## dialer-proxy

**Опционально.** Идентификатор прокси для исходящих соединений. Если заполнено, соединение OpenVPN будет установлено через этот прокси.

## remote-dns-resolve

**Необязательно.** указывает, нужно ли принудительно разрешать DNS на удалённом сервере, по умолчанию false.

## dns

**Необязательно.** работает только когда remote-dns-resolve равно true; позволяет указать список DNS-адресов.
