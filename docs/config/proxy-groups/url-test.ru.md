# Url-test

```{.yaml linenums="1"}
proxy-groups:
- name: "自动选择"
  type: url-test
  proxies:
  - ss
  - vmess
  url: 'https://www.gstatic.com/generate_204'
  interval: 300
  #tolerance: 50
  #lazy: true
```

## Общие поля

См. [Общие поля](./index.md)

## tolerance

Допустимая погрешность при переключении узлов, измеряется в миллисекундах (мс). Если разница задержек между узлами меньше tolerance, переключения не происходит — это предотвращает «скачки» между серверами с близкими пингами.

!!! tip ""
    Рекомендуемое значение — `50` (мс). Слишком маленькое значение вызовет частые переключения, слишком большое — сделает группу нечувствительной к изменениям.
